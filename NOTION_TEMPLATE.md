## Headings work like this

Use `##` at the start of a line for a section heading.

You can make words **bold** with double asterisks or *italic* with single asterisks.

Lists:
- Bullet one
- Bullet two
- Bullet three

Links look like: [link text](https://example.com)

A blockquote:
> Any line starting with a > becomes a pull quote.

Code snippets (rare for you, but here if needed):
`inline code` or

```
block of code
```
